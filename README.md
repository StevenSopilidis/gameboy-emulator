# 🟩 Game Boy Emulator (C++)

A small, slightly questionable Game Boy emulator written in C++.

## Features

* CPU emulation (mostly behaves)
* Basic graphics output
* Sound (results may vary)

## Build & Run

```bash
git clone https://github.com/StevenSopilidis/gameboy-emulator
./build.sh
./build/GameBoy
```

## Status

Works on some games. Others… build character.

## Notes

* Timing isn’t perfect
* Expect bugs
* If it crashes, that’s just the emulator expressing itself
